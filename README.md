# PDF_DEBENU_GetTextFromPDF
DebEnu PDF version 11

- PDF to Image: Generates each page to each image file (No support multipaged TIFF)

- Form Field support(add,update,delete) and Flatten PDF form fields

- support barcode
    http://www.debenu.com/docs/pdf_library_reference/DrawBarcode.php
    1 = Code39 (or Code 3 of 9)
    2 = EAN-13 or UPC
    3 = Code128
    4 = PostNet
    5 = Interleaved 2 of 5
    
- Set origin 
    0: left/buttom as (0,0)
    1: left/top as (0,0)
